    아이콘

    1. 아이콘은 fontawesome 과 비슷한 형태입니다.
    2. 아이콘 크기는 폰트사이즈에 비례합니다.
    3. 아이콘의 class는 img/icon/파일명 과 동일합니다.
    1. 누락된 아이콘이 있는 경우 전달해주세요.

    버튼

    1. 버튼은 기본적으로 활성화상태입니다.
    2. 버튼은 css 속성 또는 diabled 속성에 의해 상태가 결정됩니다.
    3. 버튼을 비활성화 하기 위해서는
      "<button class="disabled>" 또는
      "<button disabled>" 를 추가하십시오.
    1. disabled 상태일때는 클라이언트의 상효작용이 불가능합니다.
    2. 버튼 외부의 크기에 따라 버튼의 크기가 변경됩니다.

    check

    1. 체크박스 기본적으로 비활성화 상태입니다.
    2. 체크박스 checked 속성에 의해 상태가 결정됩니다.
    3. 체크박스 활성화 하기 위해서는 체크박스를 클릭하거나
      "<input type="checkbox" checked>" 또는
      "<input type="radio" checked>"
    4. 단일선택은 type="radio"
    1. disabled 상태일때는 클라이언트의 상효작용이 불가능합니다.
    2. 단일선택은 type="radio" name="group name"
    3. 복수선택은 type="checkbox"
